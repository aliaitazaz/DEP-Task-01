Task 1
  Conducting Security Audits
1.Objective: Perform comprehensive security audits for a network.
2.Description: 
    Evaluate the security posture of a network by
    identifying vulnerabilities and weaknesses. Provide
    recommendations to enhance security measures
3.Key Steps:
    Conducting a risk assessment and identifying potential threats.
    Using tools to scan for vulnerabilities.
    Reviewing security policies and procedures.
    Compiling a report with findings and recommendations.
    Presenting the audit results to stakeholders.
